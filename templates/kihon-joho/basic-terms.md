# 基本情報技術者試験 - 重要用語集

## コンピュータ構成要素

### CPU (Central Processing Unit)
中央処理装置。コンピュータの中心となる演算・制御を行う装置

### ALU (Arithmetic Logic Unit)
算術論理演算装置。加減乗除などの算術演算と論理演算を行う

### レジスタ
CPU内部の高速な記憶装置

### キャッシュメモリ
CPUと主記憶装置の速度差を埋めるための高速メモリ

### 主記憶装置
プログラムやデータを一時的に保存するメモリ（RAM）

### ROM (Read Only Memory)
読み出し専用メモリ。電源を切ってもデータが消えない

### RAM (Random Access Memory)
読み書き可能なメモリ。電源を切るとデータが消える

### DRAM
電荷による記憶方式。リフレッシュが必要

### SRAM
フリップフロップによる記憶方式。高速だが高価

### フラッシュメモリ
電気的に書き換え可能な不揮発性メモリ

## システム構成

### クライアントサーバシステム
サービスを提供するサーバと、要求するクライアントで構成されるシステム

### 3層アーキテクチャ
プレゼンテーション層、アプリケーション層、データ層の3層構造

### 負荷分散
複数のサーバに処理を分散させて負荷を軽減する技術

### 冗長化
障害に備えて予備のシステムを用意すること

### RAID
複数のハードディスクを組み合わせて信頼性や速度を向上させる技術

### RAID 0
ストライピング。高速化を実現するが冗長性なし

### RAID 1
ミラーリング。同じデータを2台に書き込む

### RAID 5
パリティを分散配置。1台の故障に対応可能

### フェールオーバー
障害発生時に予備システムへ自動的に切り替える仕組み

### フェールセーフ
故障が発生しても安全側に動作する設計思想

## ソフトウェア

### OS (Operating System)
コンピュータのハードウェアとソフトウェアを管理する基本ソフトウェア

### プロセス
実行中のプログラム

### スレッド
プロセス内の実行単位

### マルチタスク
複数のタスクを同時実行しているように見せる技術

### 仮想記憶
ハードディスクの一部を主記憶として使用する技術

### ページング
仮想記憶方式の一つ。固定長のページ単位で管理

### スワッピング
主記憶の内容を補助記憶に退避させること

### スラッシング
ページングが頻繁に発生し性能が低下する現象

### デッドロック
複数のプロセスが互いに資源を待ち合って停止する状態

### セマフォ
プロセス間の同期を取るための仕組み

### 関係データベース

### ポート番号

### ファイアウォール

### プロキシサーバ
クライアントの代理としてインターネットにアクセスするサーバ

### VPN (Virtual Private Network)
インターネット上に仮想的な専用線を構築する技術

### NAT (Network Address Translation)

### DHCP (Dynamic Host Configuration Protocol)
